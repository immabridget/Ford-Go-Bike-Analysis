Part II - Ford Go Bike Data Presentation¶

Investigation Overview¶

In this presentation I will concentrate on how the trip duration is impacted by the day of the week, hour, age and user type. These are the main features of focus.

Dataset Overview¶

Ford GoBike is a publicly available dataset that includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area.The data contains duration (secs) and others such as DateTime, customer type, gender, as well as some additional variable for each bike ride. It has 183,412 rides. The dataset can be downloaded from Here
